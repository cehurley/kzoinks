software synth

Project Summary
This project is the creation of a software synth and realtime GUI. The base system should have minimal controls, volume perhaps, but modules can be added
for individual effects. One might add an EQ module, a special type of synth module, a distortion module, etc. Each module would have defined controls that module's control box on the 
main interfdace when installed. output could be routed to DAWs or other receivers using tools like Loopback or Jackd. input should by either line in, mic or midi keyboard.
Modules can be installed by dropping self-containg folder into a directory called modules.
